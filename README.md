# 🏀 Bucket Rush

An 8-bit style basketball arcade game where you swipe to shoot hoops across multiple game modes.

<br>

## 🎮 Game Modes

<br>

| Mode | Description |
|------|-------------|
| **Classic** | Score as many baskets as you can in 60 seconds |
| **Time Attack** | Score 20 baskets as fast as possible |
| **Challenge** | 10 balls only — make every shot count |

<br>

## 🕹️ How to Play

<br>

1. Click and drag **downward** on the ball
2. Release to shoot toward the hoop
3. Score baskets before time runs out or balls run out
